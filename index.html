Qwen=client request. Lainnya alternatif.

Qwen 3 (Ollama)

ollama pull qwen3:7b

Multilingual LLM. Zero-shot. Bahasa Indonesia.

+ Zero-shot no training
+ ID language support
- 16GB+ VRAM
- ~1-3s latency

LayoutLMv3

Microsoft HuggingFace

Layout+text. Invoice, PO, receipt.

+ High accuracy visual
+ ~100ms inference
- 500+ labeled/class
